# Bend - Aplicativo de Yoga

## Sobre o Projeto Bend - Aplicativo de Yoga

Este projeto é um aplicativo de Yoga e Alongamento desenvolvido para ajudar você a se exercitar diariamente.

## Tecnologias Utilizadas

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como executar

1. Clone o repositório
2. Instale as dependências:
   ```sh
   npm install
   ```
3. Execute o servidor de desenvolvimento:
   ```sh
   npm run dev
   ```

## Funcionalidades

- Acompanhamento de sessões de yoga
- Perfil do usuário
- Preferências personalizadas
